### Pre-required
 1. PHP 8.1
 2. MySQL
 3. Laravel 10
 4. Composer 2.2.0

### Used libraries
1. Bootstrap5 (CDN)
2. jQuery (CDN)
3. openai-php/laravel

#### Don't forget to add this `.env` file:
```bash
OPENAI_API_KEY=sk-...
```
